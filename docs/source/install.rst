Setup
-----

You can install the package from PyPi:

.. code-block:: bash

    pip install bitermplus

Or from this repo:

.. code-block:: bash

    pip install git+https://github.com/maximtrp/bitermplus.git


Requirements
~~~~~~~~~~~~

* Cython
* NumPy
* Pandas
* SciPy
* Scikit-learn
* pyLDAvis (optional)
